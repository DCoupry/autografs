## What this is

AuToGraFS generates Metal-Organic Frameworks (MOFs) and other periodic crystalline structures by mapping molecular building blocks (SBUs) onto topological blueprints. Python >= 3.13. Built on pymatgen, ASE, networkx, scipy.

## Commands

```bash
pip install -e ".[dev]"          # dev install

pytest                            # run tests (slow tests skipped by default)
pytest tests/test_builder.py      # single file
pytest tests/test_builder.py -k test_name   # single test
pytest -m slow                    # run only the slow tests

black src tests                   # format (line length 88)
isort src tests                   # import sorting (black profile)
flake8 src/autografs --max-line-length=120  # lint (CI gates only on E9,F63,F7,F82)
mypy src/autografs                # type check
```

`conftest.py` auto-skips `@pytest.mark.slow` tests unless `-m` is passed explicitly.

## Architecture

Four modules in `src/autografs/`:

- **`fragment.py`** — `Fragment`: a pymatgen `Molecule` plus a `PointGroupAnalyzer`. Connection points are dummy atoms with symbol `"X"`; the symmetry analyzer describes the dummy-atom arrangement only, not the whole molecule. Fragments represent both SBUs and topology slots.
- **`topology.py`** — `Topology`: a named periodic blueprint = a pymatgen `Lattice` plus an array of `Fragment` slots. `mappings` groups symmetry-equivalent slots so one SBU choice covers all equivalent positions.
- **`builder.py`** — `Autografs`: the main entry point. Loads the SBU library (from `data/defaults.xyz` or a user XYZ file) and the topology library (from `data/topologies.pkl` via dill). `build(topology, mappings)` aligns each SBU onto its slot (Hungarian matching via pymatgen's `HungarianOrderMatcher`) while optimizing the cell parameters with Nelder-Mead, then returns a `networkx.Graph` of the periodic structure. `build_all()` enumerates every compatible SBU/topology combination.
- **`utils.py`** — XYZ parsing, UFF atom typing / force-field parameters (from `data/uff4mof.csv` and `data/covalent_radii.csv`), Fragment→MoleculeGraph/networkx conversion, GULP export.

Pipeline: SBU/topology libraries → shape-compatibility sieve (`list_building_units(sieve=...)`) → slot-to-SBU mapping → cell-scale optimization + per-slot alignment → networkx graph output.

## Gotchas

- **`data/topologies.pkl` is not in the repo.** It is generated by `scripts/cgd2pkl.py` from RCSR CGD files (`python scripts/cgd2pkl.py -o topologies.pkl --use_rcsr`). Tests that need it check availability via the `data_files_available` fixture; `Autografs()` will fail to load topologies without it.
- **README.rst examples describe the old 2.x API** (`make()`, `functionalize()`, `flip()`, `mmanalysis`...). The current 3.0 API is `build()`, `build_all()`, `list_topologies()`, `list_building_units()`. The README notes old functionality is not yet reimplemented — don't treat its snippets as the current interface.
- Dummy atoms use pymatgen's `"X"` species convention throughout; slot/SBU compatibility is decided by dummy count and point group.
